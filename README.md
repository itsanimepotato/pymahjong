To win, make 4 triplets and a pair

A triplet = [1][1][1] (3 of a Kind) OR [1][2][3] (Sequence)

A pair = [1][1]

To make a set, you can steal from other people if it completes the set:
Pung – 3 identical tiles

Kong – 4 identical tiles

Chow – a sequence of 3 in the same suit

## Flow of the Game
- at the start of the game: roll 2 dice to find who is the east wind (dealer for that game)
- deal 13 tiles to everyone
- replace bonus tiles with a new tile and set bonus tile aside (extra point)
- the starting wind (CCW of last round, if first round then east starts) gets an extra tile (total is 14 tiles) and discards one to start the game
- order goes CCW, each person getting a tile and discarding a tile until they make 4 triplets and a pair
